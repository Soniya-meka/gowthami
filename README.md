# Gowtham Institutions Website

A premium, modern school website developed with HTML, CSS, and JS, powered by Vite.

## Features
- **Modern UI**: Clean, responsive, and aesthetic design.
- **Custom Visuals**: High-resolution imagery for all school facilities.
- **Fast Performance**: Built with Vite for optimal loading.

## Development
To run this project locally:
```bash
npm install
npm run dev
```

## Deployment
This project is ready to be deployed to **Vercel**. 
The build settings should be:
- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
